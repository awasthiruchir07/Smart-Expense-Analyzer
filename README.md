# Smart Expense Analyzer

## Overview
This is a Command-Line Interface (CLI) based expense tracker that uses basic AI techniques to analyze spending habits. It stores data in CSV format and works completely offline.

## Features
- Add, view, and manage expenses
- Automatic expense categorization
- Spending analysis and reports
- Basic prediction of future expenses

## Technologies Used
- Python
- CSV (for storage)
- Standard libraries (datetime, statistics)

## How to Run
1. Install Python
2. Run the program:
   python PythonAI.py

## File Structure
- PythonAI.py → Main application
- expenses.csv → Data storage

## Use Case
Helpful for students to track daily expenses and understand spending behavior.
